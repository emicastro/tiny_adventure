## How to use it?

- Open a terminal
- Clone the repository: `git clone git@github.com:emicastro/tiny_adventure.git`
- `cd tiny_adventure`
- `cargo run`
- Have fun and be free to modify it and make your own game!

## Requirements

- Have `rust` with `cargo` installed in your system.
- Access to a terminal.
- Time to play.
